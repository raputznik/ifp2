== Description ==
Easily add an option under user's personal option to show/hide toolbar. In wordpress 3.3, the Header + Admin Bar = Toolbar thats why the naming covention for this plugin. 

Adds an option to the User profile edit screeen to "Show Toolbar when viewing admin" in addition to wordpress 3.3's option to " Show Toolbar when viewing site"

This plugin is tested only on the latest 3.3 and will be tested against all future releases. Its more of an add on to the wordpress 3.3 toolbar hide functionality. 

= Features: = 
Options Page that allows you to:
* Show Toolbar when viewing admin 
* Change setting via "Show Toolbar when viewing admin" in user profile edit screen.

== Installation ==

= If you downloaded this plugin: =
* Upload `hide-toolbar` folder to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Once Activated, the toolbar from wordpress admin screen will be gone. 
* If you want to override the settings, simply override it by  going to your user profile's edit screen and check the "Show Toolbar when viewing admin" option under Personal Options.

= If you install this plugin through WordPress 2.8+ plugin search interface: =
* Click Install `Hide Toolbar`
* Activate the plugin through the 'Plugins' menu in WordPress
* Once Activated, you can add access the options page from your user profile's edit screen and check the "Show Toolbar when viewing admin" option under Personal Options.

== Frequently Asked Questions ==
= How does it work? =
It simply hides your wordpress toolbar from admin. Wordpress 3.3 already has an option of hiding the toolbar from the front end i.e from website view.

= Will it effect only me or will it hide the toolbar for other users on the site =
It only effects your profile. All other users will still be able to adjust their settings.
